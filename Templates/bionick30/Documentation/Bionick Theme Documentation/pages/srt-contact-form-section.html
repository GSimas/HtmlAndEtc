Contact Form Section Shortcode

Step 1: Get/Create Contact Form 7 ID

  1. Add this code at contact form 7 content area.
    2. <p>[text* your-name watermark "Name"]</p>

    
<p>[email* your-email watermark "E-mail"]</p>

    
<p>[text your-subject watermark "Subject"] </p>

    
<p>[textarea* your-message watermark "Message"] </p>

    
<p>[submit "Send"]</p>
  2. Copy the contact form 7 ID number.

Step 2: Generate Contact Form Shortcode

This shortcode is for add contact form section area which has options for add extra and contact form 7 ID.

  1. To generate this shortcode click Add Element or (+) → Bionick Contact Form
  2. Insert data from Bionick Contact Form Settings by visual options.
  3. Note: Required Contact Form 7 WordPress Plugin Installed/Activated
