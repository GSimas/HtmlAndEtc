Theme Installation

Firstly thanks for purchasing! We're so glad you chose our theme and hope it satisfies your needs! The steps below assume that you already have WordPress installed, if you don't go here for a WordPress installation guide.

Method 1: Via The WordPress Dashboard (Easier)

  1. Unzip the package you downloaded from ThemeForest & inside you will find a folder called main_file_bionick and inside this you will find a file called bionickwp.zip
  2. Login to your WordPress dashboard and go to Appearance > Themes click on the Install Themes tab and select Upload
  3. Click the Browse button and find the bionickwp.zip file
  4. After the theme has successfully uploaded click Activate and you're done!

Method 2: Via FTP

  1. Unzip the package you downloaded from ThemeForest & inside you will find a folder called main_file_bionick and inside this you will find a folder called bionickwp
  2. Login to your web server via FTP and browse to wp-content/themes and upload the bionickwp folder here
  3. Login to your WordPress dashboard and go to Appearance > Themes the BionickWP theme will be visible, click Activate and you're done!
