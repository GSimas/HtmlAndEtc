Portfolio Page Setup

  1. Go to Pages > Add New
  2. Add your Page Title in the title field
  3. Select Portfolio Page Template from the Template selection on the right side of the page
  4. Click Publish

Page Settings

  1. Section Title: To enable/disable section title at the header select Enable or Disable
  2. Section Title Text: To add section title text at the left side of the page write down at this text field.
  3. Section Subtitle Text: To add section subtitle text at the header write down at this text field.
  4. Open as a separate page: Check it while using "One Page Menu" & want to open this page as a separate page.

Portfolio Page Settings

  1. Portfolio Page Layout: Bionick has three types Portfolio page layout style and they are Default | Inner | Full Width   page style.
  2. Portfolio Filter: Bionick has three types Portfolio Filter Category and they are Portfolio | Portfolio 2 | Portfolio 3   which is for 3 different filterable portfolio page.
  1. Upload Section Background Image for the default & inner portfolio page layout style from the featured image option on the right side of the page
