Portfolio Post Setup

Bionick has three types Portfolio Filter Category and they are Portfolio | Portfolio 2 | Portfolio 3   which is for 3 different filterable portfolio page.

  1. Go to Portfolio → Add Add New Portfolio Item
  2. Add your portfolio title in the title field.
  3. Add your portfolio content in the text area.
  4. Upload the portfolio image by Featured Image

Portfolio Options

  1. Image Size: To select image size for the portfolio page.
  2. Icon Name: To add Font Awesome icon name for the portfolio page at thumbnail image hover effect style.
  3. Post Formats: Bionick has four types post format style and they are Image | Slider | Gallery | Video
  4. Slider/Gallery Images: For the "Gallery or Slider" portfolio post format add images by click upload images.
  5. Video Player: For the "Video" portfolio post format choose video player YouTube | Vimeo
  6. YouTube/Vimeo Video Link: For the "YouTube/Vimeo" portfolio post video player to add video by insert youtube/viemo video embed url link. Example:
    YouTube → https://www.youtube.com/embed/RRPU_mfhYGA
    Vimeo → //player.vimeo.com/video/88131286

Portfolio Section

  1. Section Title: To add section title for the portfolio details page.
  2. Section Style: Bionick has four types section style for the portfolio details page and they are Image | Slider | Default | Video
  3. Section Images: For the "Default or Image or Slider" section style add images by click upload images.
  4. Youtube Video ID: For the "Video" section style add video by insert youtube video ID.
  5. Section Video Sound: To enable/disable video sound select Enable or Disable
