Blog Post Setup

  1. Go to Post → Add New Post
  2. Add your post title in the title field.
  3. Add your post content in the text area.
  4. Image post format upload the image by Featured Image
  5. Video post format insert youtube/viemo video embed url link by Post Options. Example:
    Youtube → https://www.youtube.com/embed/RRPU_mfhYGA
    Viemo → //player.vimeo.com/video/88131286
  6. Audio post format insert soundcloud audio embed url link by Post Options
  7. Gallery post format upload images by Post Options
