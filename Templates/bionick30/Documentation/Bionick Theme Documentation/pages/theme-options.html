Theme Options

Bionick comes with a theme options panel which allows you to easily customize your website, to access it go to WordPress Dashboard Navigations named by Bionick Options and click Save Changes to save. And click Reset Section to reset changes.

General Settings

  1. Site Logo: To add site logo image click upload button.
  2. Logo Padding: To adjust site logo image position by using padding.
  3. Favicon Upload: To add favion image click upload button.
  4. Portfolio Page Link: To add portfolio page link which need for portfolio details page.

Blog Settings:

  1. Layout Style: To choose layout style for blog details, archives, category, tag & search page. Select Default or Right Sidebar or Left Sidebar options
  2. Section Image: To add section image for blog "Default" layout style click upload.
  3. Section Title Text: To add section title for blog "Default" layout style write down at this text field.
  4. Header Search Box: To show/hide header search box at the top of the header for blog details, archives, category, tag & search page. Select Enable or Disable button
  5. Page Title: To show/hide page at the header for blog details, archives, category, tag & search page. Select Enable or Disable button
  6. Page Title Text: To add page title for blog details page write down at this text field.
  7. Page Subtitle Text: To add page subtitle for blog details page write down at this text field.
  8. Post Social Share: To show/hide post social share icon at blog details page. Select Enable or Disable button
  9. Share Text: To add share icon text for blog details page write down at this text field.
  10. Posted By Text: To change/repalce blog post "Posted By" text write down at this text field.
  11. Continue Reading Text: To change/repalce blog post "Continue reading ..." text write down at this text field.
  12. Archive Page Title: To change/repalce blog post "Archive" page title text write down at this text field.
  13. Category Page Title: To change/repalce blog post "Category " page title text write down at this text field.
  14. Tag Page Title: To change/repalce blog post "Tag " page title text write down at this text field.

Styling:

  1. Theme Color: Bionick has unlimited color option. To change site theme color select from here.
  2. Custom CSS: To add/edit CSS code write down the codes at this text area.

Typography:

Bionick has typography option with each property can be called individually.

Footer Settings:

  1. Copyright text of the WebSite: To add copyright content at the footer area write down at this text area.
