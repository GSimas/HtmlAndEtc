Blog Page Setup

  1. Go to Pages > Add New
  2. Add your Page Title in the title field
  3. Select Blog Page Template from the Template selection on the right side of the page
  4. Click Publish

Page Settings

  1. Section Title: To enable/disable section title at the header select Enable or Disable
  2. Section Title Text: To add section title text at the left side of the page write down at this text field.
  3. Section Subtitle Text: To add section subtitle text at the header write down at this text field.
  4. Open as a separate page: Check it while using "One Page Menu" & want to open this page as a separate page.

Blog Page Settings

  1. Blog Page Layout: Bionick has three types blog page layout style and they are Default | Right Sidebar | Left Sidebar   page style.
  2. Number Of Post Show: To set number of post show at the blog page by slider option
  3. Exclude Category: To show posts from specific category by insert a category name
  1. Upload Section Background Image for the default blog page layout style from the featured image option on the right side of the page
  2. Click Set featured image

Page Intro Section Settings

  1. Intro Section: To enable/disable intro section select Enable or Disable
  2. Intro Section Title: To add intro section title text write down at this text field.
  3. Intro Section Subitle: To add intro section subtitle text write down at this text field.
  4. Intro Section Style: Bionick has five types intro section style and they are Image | Slider | Slideshow | Video | Revolution Slider
  5. Intro Section Images: To add image for Image intro section style click to upload images.
  6. Slider/Slideshow Category: To add slider images for Slider intro section style insert slide category name.
  7. Youtube Video ID: To add video for Video intro section style insert youtube video ID.
  8. Video Sound: To enable/disable video sound select Enable or Disable
  9. Revolution Slider ID: To add revolution slider images for Revolution Slider intro section style insert revolution slider shortcode.
