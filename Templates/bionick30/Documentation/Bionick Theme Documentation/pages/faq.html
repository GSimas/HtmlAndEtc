You can change logo background color yellow to any color you like by adding CSS codes.

Here you go: Bionick Options > Styling > Custom CSS || And add this CSS codes at the textarea. 

.logo-holder {
    background: #191919!important;
}

Note: You can replace/change 191919 this color code to any color code you like.

That means you can create three different filterable portfolio page. But you can't create dynamically/unlimited new filterable portfolio categories, right now it's unavailable.

Portfolio post with multiple categories is unavailable right now. That means you can't add multiple categories to a single portfolio post (one portfolio post = one category).
