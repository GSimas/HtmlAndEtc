Resume Post Setup

  1. Go to Resume → Add New Resume
  2. Add your resume title in the title field.
  3. Add your resume content in the text area.
  4. Icon: To add Font Awesome icon name by Resume Options
  5. Time Duration: To insert occupation time duration by Resume Options
  6. Main Category: To insert occupation main category or subtitle text by Resume Options
  7. Sub Category: To insert occupation sub category names or list by Resume Options
