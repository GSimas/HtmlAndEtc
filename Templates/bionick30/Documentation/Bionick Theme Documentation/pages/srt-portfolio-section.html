Portfolio Shortcode

Bionick has three types Portfolio Filter Category and they are Portfolio | Portfolio 2 | Portfolio 3   which is for 3 different filterable portfolio page. For that there is three portfolio shortcodes for each portfolio filter and they are Bionick Portfolio | Bionick Portfolio 2 | Bionick Portfolio 3

This shortcode is for portfolio post section area which has options for add extra class.

  1. To generate this shortcode click Add Element or (+) → Bionick Portfolio
  2. Insert data from Bionick Portfolio Settings by visual options.
