Import Dummy Data

Some dummy data has been included to help you get started, its not identical to the demo site but hopefully it'll give you a general idea of how things work. If you run into any problems or have any question or need any instruction/help to work with this theme feel free to ask our support team and we'll be happy to help! Please send us mail at our support via envato.

Step 1: Demo Dummy Content

  1. Install the WordPress Importer Plugin
  2. Now go to Tools > Import > WordPress and select the import-data.xml file from the Dummy Content folder.
  3. Go to Appearance > Menus for the Multi-Page Site select Primary Menu and for the One Page Site select One Page Menu.
  4. Go to Settings > Reading > Front page displays > Front page: Select Intro page for one page site.

Step 2: Demo Dummy Widget

  1. Install the WordPress Widget Importer & Exporter Plugin
  2. Now go to Tools > Widget Importer & Exporter > Import Widgets and select/choose the import-widget-data.wie file from the Dummy Content > Widget Data folder.
